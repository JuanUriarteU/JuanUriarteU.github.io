E3 Mecanizado 2.5 D

Ejemplo de Mecanizado CNC 2.5D para la Silla Fresia

Objetivo General

  1. Comprender el manejo de las herramientas básicas del software Fusion 360 para el uso de mecanizado en router 2.5D, donde extraeremos un respaldo.

Objetivo Específicos

  1. Entender el funcionamiento de cortes, desbaste, seteo de Máquina, de fresa y direcciones de cortes.
  2. Uso de Drawings en Fusion 360.
  3. Uso de Manufacture - Adaptative Clearing
  4. Uso de Manufacture - Parallel

¿Qué es el Mecanizado 2.5D?

Se refiere a los usos de los ejes en el mecanizado, ya que la máquina se puede mover libre y simultanemente por los ejes X e y mientras que en el eje Z solo se permite un valor único, mientras trabaja. Donde el eje Z se setea a cierta altura y el router se mueve libremente por los otros ejes para luego volver al punto inicial para poder cambiar la altura nuevamente. Esto no se llama 3D, ya que solo puede mover dos de los 3 ejes (x e y ) de manera libre, mientras que si fuese 3D podría cambiar la altura al mismo tiempo que va trabajando.

Este mecanizado se usa para el desbaste de grandes cantidades de material, de manera expedita en piezas fáciles de cortar.

Si comparamos el acabado de un mecanizado 3D vs 2.5D, el 3D saldría ganando debido a que entrega mayor calidad y este permite también realizar acabados de geometrías más complejas, donde suele realizarse el desbaste en 2.5D y luego para una mejor terminación se realiza en 3D.

Proceso de Mecanizado

En primer lugar debemos seleccionar la pieza que se desea mecanizar, donde la cual en este caso seleccionamos el respaldo, para luego abrirlo como un nuevo archivo y posterior a eso alinearlo con los ejes, bien cerca del origen, para poder llevar un uso correcto de la máquina y no toparnos con contratiempos.

De esta manera seleccionamos una cara y se seleccionamos Modify y luego ALING.

Luego de haber ubicado correctamente la pieza, tengo que diseñar y plasmar el área donde voy a trabajar, por lo que genero un sketch, la cual será la plancha de donde se extraerá la pieza.

Luego de crear este sketch se extruye para poder cubrir todo el respaldo, esto se hace mediante la seleccion de la superficie y luego un extrude (e), donde luego se le baja la opacidad al paralelepipedo que queda para poder ver el objeto a extraer.

Ahora terminamos la parte de design y nos metemos a la parte de manufacture para poder hacer el set up, donde selecionaremos el tipo de máquina, en este caso la Generic 3-Axis. Luego de ello, se define el stock, y mode se selecciona from Solid y le damos click a la plancha que extruimos anteriormente. Ahora seleccionamos Orientation donde seleccionamos Select Z axis / Plane & X axis, donde primero seleccionamos la cara superior del paralelepipedo, luego la arista superior del eje X y después Stockpoint y ponemos Box point donde selccionamos el vertice superior izquierdo del mismo lado del origen, y finalmente en model seleccionamos el respaldo, con esto ya tenemos el setup ready!

Teniendo listo el setup proseguimos a hacer otro sketch donde haremos un rectangulo proyectado para delimitar el área donde se optimizará el mecanizado para que la máquina no trabaje de más, esto lo hacemos en design y selecionamos la tecla P y después el respaldo.

De esta manera tenemos todo en posición para empezar a realizar el mecanizado, el cual se hará por ambos lados de la pieza, esto se hará en dos partes, parte superior e inferior.

La primera operación que se realizará es “Adaptative Clearing” el cual elimina material de una forma rápida sin terminaciones muy precisas, nos sirve para dar un cepillado a la primera forma y obtenerla rápidamente optimizando el tiempo. Seguimos con la fresa flat end mill y la generic 3-axis, la cual debo seleccionarla nuevamente.

De manera siguiente para mejorar las terminaciones, se realiza el parallel para darle una terminación más limpia y no tan bruta como la realiza en el Adapatative Clearing, por otro lado también cambiamos la fresa a una de punta redonda para alcanzar mejor esta terminacion. De manera consecutiva le ordeno a la máquina que haga ese mismo ejercicio pero en dirección perpendicular a la anterior, por lo que duplico la operación pero edito el pass direction y lo cambio a 90° para que se haga en la otra direccion y tambien cambio el stepover a 2 para que sea mejor la resolucion de la terminacion.
