Mecanizado CNC - Silla Fresia

Ejemplo de Mecanizado CNC para la Silla Fresia

Objetivo General

  1. Comprender el manejo de las herramientas básicas del software Fusion 360 para el uso de mecanizado en router CNC.

Objetivo Específicos

  1. Entender el funcionamiento de cortes, seteo de Máquina, de fresa y direcciones de cortes.
  2. Uso de Drawings en Fusion 360.
  3. Uso de Contours.
  4. Uso de Pocket.
  5. Uso de Trace.
  6. Uso de Tabs
  7. Entender los software de diseño 3D desde una perspectiva de fabricación.

Videos Del Proceso

En primer lugar tomamos las distintas partes de la silla y las colocamos de manera ordenada y consecutiva en media plancha de terciado, para luego realizar los distintos cortes

Luego seleccionamos una fresa para que haga los cortes.

Luego dimos la orden para hacer contours en los sacados de las sillas para poder ensamblar las distintas piezas.

Más tarde tuvimos que hacer sacados llamados Pocket,en los cuales van en las intersecciones donde se juntan las distintas piezas para que así encajen.

Luego se hacen nuevamente contours, pero esta vez para cortar la pieza completa, en esta ocasión se hacen secciones de tabs para que la pieza no se desprenda al tiritar por la vibración de la máquina. Es decir, se mantendrá unida a la plancha por pequeñas secciones de madera.

A continuación una simulación de el proceso completo del corte y su recorrido.

Por último, para el asiento se ejecutó un tipo de calado llamado Trace el cual permite darle curvatura a la madera, lo cual se logra mediante pequeños cortes en la madera.
